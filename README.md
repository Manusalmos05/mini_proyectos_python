# 🐍 Miniproyectos Python

> **Aprendiendo Python un proyecto a la vez.** 🚀

Bienvenido a este repositorio, donde recopilo los ejercicios, retos y miniproyectos que desarrollo durante mi formación en Python como parte del **IBM Curso Certificado de Python Avanzado orientado al Desarrollo Backend**.

Mi objetivo no es únicamente completar un curso, sino **construir una base sólida de programación**, desarrollar buenas prácticas y crear un historial de aprendizaje que refleje mi evolución como futura profesional del análisis de datos.

---

## 🎯 Objetivos del repositorio

- Practicar los fundamentos de Python.
- Resolver problemas mediante programación.
- Mejorar mi capacidad de análisis y pensamiento lógico.
- Aplicar buenas prácticas de codificación.
- Documentar mi progreso de forma transparente.
